# Kontrollfragen 1

## Eine Liste mit mindestens 6 IT-Trends nennen & erläutern
1. Künstliche Intelligenz (KI):
    - Maschienen und Software, die lernen, denken und Entschiedungen treffen können
    - Bsp.: Chatbots, Bilderkennung
2. Cloud Computing:
	- Bereitstellung von IT-Resourcen (Server, SPeicher, Software)
	- Bsp.: AWS, Azure
3. Edge Computing
    - Datenverarbeitung nahe am Ort der Datenverwendung statt zentral in der Cloud
    - Bsp.: Echtzeit-Analyse
4. Internet of Things (IoT):
	- Vernetzung von Geräten & Sensoren über das Internet
	- Bsp.: Smart Home, vernetzte Autos
5. Cybersecurtiy / IT-Sicherheit:
    - Schutz von Daten, Netzwerken & Systemen
    - Bsp.: Firewalls, Verschlüsselung, MFA
6. Blockchain:
    - Dezentrale, fälschungssichere Datenbankentechnologie, die Transaktionen transpanent speichert
    - Kryptowährungen (Bitcoin, Ripple, XRP, etc.)

## Unterschied zwischen Edge- und Cloud-Computing
### Cloud Computing
 - Defintion: Bereitstellung von Rechenleistung, Speicher und ANwendungen über das Internet in zentralen Rechenzentren
 - Ort der Verarbeitung: Daten werden in entfernten Servern (CLoud) verarbeitet
 - Vorteile:
    - Sklaierbarkeit
    - Kosteneffinzient
    - Zugriff von Überall möglich
 - Nachteile:
    - Latenz
 - Bsp.: Amazon EC2 (virtuelle Server), S3 (Cloud Speicher)
### Edge Computing
 - Defintion: Verarbeitung von Daten direkt am Ort der Datenerzeugung
 - Ort der Verarbeitung: Nahe an Sensoren, Maschienen oder Endgeräten
 - Vorteile:
    - geringe Latenz
    - Entlastung des Netzwerks
 - Nachteile:
    - lokale Geräte müssen Leistungsfähig sein
    - Skalierung oft schwieriger als in der Cloud

`Cloud-Computing verarbeitet Daten zentral in der Cloud, während Edge-Computing Daten dezentral an Entstehungsort verarbeitet Daher ist Edge schneller für Echtzeit-Anwendungen und Cloud besser für Skalierung und große Datenmengen`
