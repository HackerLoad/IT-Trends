## AWS CLI zeigen

1. AWS CLI installieren (falls nicht vorhanden):
   - Windows: MSI Installer
   - Linux/Mac:
     curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
     unzip awscliv2.zip
     sudo ./aws/install
2. Terminal öffnen
3. Befehl testen:
   - aws --version
4. Befehl ausführen und Daten aus AWS einfügen:
   - aws configure
